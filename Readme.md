1. Ex1: Get largest component in a graph
2. Ex2: check if graph has a cycle
3. Ex3: count the islands in a grid

For instance we may have a grid defined as follow:
```
grid = [
    ["W", "L", "W", "W", "L", "L"],
    ["W", "W", "W", "W", "W", "W"],
    ["W", "W", "W", "W", "L", "W"],
    ["W", "L", "L", "W", "L", "L"],
    ["W", "L", "L", "W", "L", "W"],
    ["W", "W", "W", "W", "W", "W"],
]

```

where `"W"` stands for water, and `"L"` for land.
